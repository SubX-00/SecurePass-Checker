# SecurePass Checker

![SecurePass Checker Screenshot](https://github.com/SubX-00/SecurePass-Checker/blob/main/screenshot%20(2).png?raw=true)

**SecurePass Checker** is a modern and interactive password strength checker built with **HTML, CSS, and JavaScript**. It gives **real-time feedback** on password strength and tells the user exactly what’s missing to make a secure password.

---

## Features

- **Live Password Strength**  
  A colored bar shows Weak → Medium → Strong passwords as you type.

- **Detailed Hints**  
  Users are informed if the password is missing:
  - Minimum 8 characters
  - Uppercase letters
  - Numbers
  - Special characters (e.g., `@`, `!`, `#`)

- **Show/Hide Password Toggle**  
  Easily toggle between hidden and visible password input.

- **Responsive Design**  
  Works beautifully on desktop, tablet, and mobile devices.

---

## How to Use

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Enter a password in the input field.
4. Observe the **strength bar** and **feedback message**.
5. Use the 👁️ button to toggle password visibility.

---

## Example

- Weak password: `abc` → Shows red bar + feedback:  
  `Weak Password ❌. Add: at least 8 characters, an uppercase letter, a number, a special character like @`

- Strong password: `P@ssW0rd123` → Shows green bar + feedback:  
  `Strong Password ✅`
